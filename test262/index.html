LibJS test262 results

These are the results of the SerenityOS JavaScript engine, LibJS, running the official ECMAScript conformance test suite test262 as well as the test262 parser tests. All tests are re-run and the results updated automatically for every push to the master branch of the repository on GitHub. Dates and times are shown in your browser's timezone. Development of the engine started in March 2020 (see this YouTube video), continuous running of test262 in June 2021 (see this PR on GitHub) — results before that time are not available.

The source code of the LibJS test262 runner and aggregation script is available here, the source code of this website here, and the raw data here (JSON).

If you have any questions or want to help out with improving these test scores, feel free to get in touch on the SerenityOS Discord server (#js).
